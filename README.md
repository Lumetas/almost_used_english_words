# Идея
Словарный запас не самое главное в языке, но не зная слов граматика не поможет, это не большой проектик скиданный нейронкой, предостовляющий интерактивные карточки дабы эффективно выучить 1000 самых популярных английских слов, размещён на github pages по [ссылке](https://lumetas.github.io/almost_used_english_words/)


В репозитории также есть файл words.json, из которого и берутся слова. Вы можете взять его из файла, по прямой ссылке:
```
https://raw.githubusercontent.com/Lumetas/almost_used_english_words/refs/heads/main/words.json
``` 
Если вам это нужно, извините за качество, проект был сделан для личного использования и сделан под меня
